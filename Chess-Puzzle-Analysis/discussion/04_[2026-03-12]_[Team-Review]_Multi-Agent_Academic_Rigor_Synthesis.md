## 1. 팀 합의 사항 (Consensus Findings)

4개 리뷰어가 **독립적으로 동일한 결론에 도달한** 핵심 사항들:

### 1.1 주장 수준 재조정 필요 (4/4 합의, 최우선)

| 리뷰어 | 표현 |
|--------|------|
| A | "탐색적 연구로서의 가치는 충분히 있으나, 확증적(confirmatory) 프레이밍은 정당화되지 않음" |
| B | "이 근거 수준은 탐색적 이중 데이터셋 연구로서 적절히 기대되는 수준" |
| C | "사후적 이론 맞추기(post-hoc pattern matching)" 구조가 텍스트 증거로 입증됨 |
| D | "Chess 결과를 'suggestive'로, 명시적 소표본 주의사항 포함" |

**팀 결정:** "5개 이론 패턴이 data-driven으로 확인되었다" → "탐색적 GMM에서 이론과 일관된 클래스 구조가 관찰되었으며, 향후 확인적 연구의 가설 기반을 제공한다"로 전환. **이것만으로 가장 치명적인 비판의 위력이 크게 감소한다.**

### 1.2 LCGA/GBTM 비교 분석 필수 (4/4 합의)

| 리뷰어 | 표현 |
|--------|------|
| A | "mclust vs lcmm 결과 수렴 시 비판 #9 무력화 + 과적합 우려 간접 완화" |
| B | "Oscillating Low 등급 → 100+ 시행 데이터 + LCGA로 상향 가능" |
| C | "파이프라인 독립성 확보 위해 전혀 다른 방법론 적용 필요" |
| D | "LCGA를 primary trajectory method로, GMM을 supplementary로 격상/격하" |

**팀 결정:** R `lcmm` 패키지로 LCGA 분석 실행. Chess에서 piecewise linear(전환점 W4.5), EdNet에서 quadratic model. **우선순위 1.**

### 1.3 N=100에서 6-클래스 과적합 심각 (4/4 합의)

| 리뷰어 | 세부 |
|--------|------|
| A | 파라미터 151개 vs 관측치 100개 = 비율 0.66:1 |
| B | Catastrophic N=30, ABE N=9, Class 6 N=2 — 소표본 한계 |
| C | 규칙 기반 임계값이 과적합 위에 추가적 자유도 부여 |
| D | 문헌 기준 N≥200 권장, BIC는 N=100에서 50-70% 정확도 |

**팀 결정:** (1) PCA로 차원 축소 후 GMM 재실행, (2) SABIC + ICL + BLRT 다중 기준 보고, (3) Class 6(N=2) 결과 해석 시 명시적 주의사항 포함.

### 1.4 AI Benefit Emergence의 HARKing 구조 인정 필요 (3/4 합의, A+B+C)

| 리뷰어 | 판단 |
|--------|------|
| A | 순환논증 — EdNet에서 발견 후 이론 추가, Chess에서 "예측 확인" 주장 |
| B | "탐색적 발견 특성"이 "높음" 등급 진입을 차단하는 주요 요인 |
| C | "가장 명백한 HARKing 지점" — 텍스트 증거 있음 |
| D | (직접 언급 없음 — 방법론 적합성에 집중) |

**팀 결정:** AI Benefit Emergence를 "EdNet 분석에서 귀납적으로 발견된 신규 패턴"으로 투명하게 재분류. 나머지 4개 패턴은 "사전 이론적 예측", ABE는 "탐색적 발견"으로 명확히 구분. **이것이 약점이 아닌 이론적 기여(귀납적 이론 확장)로 재프레이밍 가능.**

## 3. 리뷰어 간 쟁점 및 해소 (Points of Divergence)

### 3.1 Gap ≈ −0.600 수렴의 해석

| 리뷰어 A | "산술적 귀결 — R_b≈0.20, AI≈0.80이면 자동으로 −0.60" |
|----------|------|
| **리뷰어 B** | "방향적 일관성은 증거이나, 수치 일치 자체를 '놀라운 재현'으로 해석하면 안 됨" |
| **리뷰어 C** | "파이프라인 동일성에 의한 인공물 가능성 배제 필요" |

**팀 결정:** Gap 수치 일치를 "재현 증거"가 아닌 "동일 패턴 유형의 수치적 특성"으로 재프레이밍. 시뮬레이션으로 null distribution 확인 권고하되, 방향적 일관성(두 데이터셋 모두에서 극단적 under-reliance 그룹 존재)은 유의미한 발견으로 유지.

### 3.2 Cohen's d = 2.0의 의미

| 리뷰어 A | d=2.0의 대부분은 AI_accuracy 조작에 의한 산술적 귀결, 순수 행동 변화는 d≈0.60 |
|----------|------|
| **리뷰어 C** | "실험 조작 확인(manipulation check) 수준" |
| **리뷰어 D** | (직접 언급 없음) |

**팀 결정:** (1) R_b 변화의 효과 크기(d≈0.60)를 별도 보고, (2) cal_gap d=2.0이 "조작량 + 적응 실패"의 합성임을 분해 제시, (3) appropriate_reliance 효과 크기를 cal_gap보다 우선 보고.

### 3.3 조건별 GMM의 충분성

| 리뷰어 A | 조건별 GMM + bimodality 검정(Hartigan's dip test) 필수 |
|----------|------|
| **리뷰어 D** | 조건별 GMM 시 N=50에서 G는 2-3으로 제한 권장 |

**팀 결정:** `chess_gmm_by_condition.R` 이미 실행 완료. C2 부트스트랩 불안정성(G=5가 17%만) 문제를 명시적으로 보고. Hartigan's dip test 추가 실행 권고.

## 6. 연구의 강점 확인 (팀 합의)

비판에 집중했지만, 4개 리뷰어 모두 인정한 강점:

1. **이론적 프레임워크의 가치**: T × R × τ 3차원 신뢰 보정 역학 모형은 기존 문헌에 없는 독창적 기여 (B, D)
2. **이중 데이터셋 설계**: 서로 다른 도메인(교육 vs 체스), 다른 설계(관찰 vs 실험)에서의 교차 확인은 탐색적 연구로서 드문 강점 (A, B)
3. **Chess에서의 Catastrophic 패턴**: C1 참가자 60%의 과의존 지속은 automation complacency 문헌과 직접 연결되는 명확한 발견 (B)
4. **투명한 자기 인식**: Discussion 02에서 리뷰어 비판 예측과 사전등록 부재 인정 (C)
5. **Chess CC BY 4.0 데이터 권한**: IRB 면제 가능, 재현 가능한 분석 파이프라인 (D)
